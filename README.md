## Project Overview

Travlr Getaways is a full stack web application developed throughout the CS-465 Full Stack Development I course. The project begins as a static website and is progressively enhanced into a complete web application using modern web development technologies and the Model-View-Controller (MVC) architecture. Throughout the course, additional functionality, routing, templating, data management, and backend features are implemented as part of each module.
